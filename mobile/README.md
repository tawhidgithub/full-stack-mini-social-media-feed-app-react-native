# Mini Social Feed App 📱

A modern social media mobile app built with **React Native (Expo)** and **TypeScript**, featuring posts, likes, comments, infinite scroll, and push notifications.

---

## 🚀 Features

- 🔐 Authentication (Login / Register)
- 📝 Create & view posts
- ❤️ Like / Unlike posts (optimistic UI)
- 💬 Comment system
- 🔍 Search posts by username (server-side)
- 📜 Infinite scroll pagination
- 🔔 Push notifications (Expo Notifications)
- ⚡ Real-time-like UX with optimistic updates

---

## 🧰 Tech Stack

- React Native (Expo)
- TypeScript
- Axios
- React Navigation
- AsyncStorage
- Expo Notifications

---
